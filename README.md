# 📚 Projects

## ✅ Todo Backend API

A secure RESTful Todo Management API built with Flask using the **App Factory Pattern** and **Service Layer Architecture**.

### Features

* JWT Authentication
* User Registration & Login
* User Profile Endpoint
* CRUD Operations for Todos
* User-specific Todo Management
* Pagination
* Filtering by Completion Status
* Protected Routes
* Input Validation
* Structured JSON Responses

**Tech Stack**

* Flask
* Flask-SQLAlchemy
* Flask-Migrate
* Flask-JWT-Extended
* SQLite
* Werkzeug
* python-dotenv

📄 See the complete documentation in:

```text
todo-backend/README.md
```

## 📖 Book Catalog API

A RESTful Book Catalog API for managing a library's collection with **role-based authorization**.

* CRUD Operations for Books
* Search Books

* Book Statistics
* Role-based Authorization
